## Steps

- Tell the user we are starting Phase 8.
- In Jsonmaker:
  - Select top-level node to sync
  - Click **View Node**
  - Copy View Node URL
- In browser:
  - Open Subscribed Toolbar **Options**
  - Paste URL into **Feed URL**

  - Click **SYNC**
- **VERIFY ORDER:** `SAVE` -> `SYNC`
